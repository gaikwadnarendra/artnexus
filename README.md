# 🎨 ArtNexus

ArtNexus is a web-based platform designed to showcase and explore digital art. It allows users to browse, upload, and interact with artwork in a clean and modern interface.

🌐 Live Demo: https://artnexus.onrender.com/

---

## 🚀 Features

- 🖼️ Explore digital artworks
- ⬆️ Upload and share your own art
- 👤 User-friendly interface
- 📱 Responsive design (mobile + desktop)
- 🔍 Easy navigation and browsing

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Django (Python)

**Database:**
- Postgres

**Hosting:**
- Render
